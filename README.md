# GoGold — Gold Detector Recommender

A simple static site that recommends gold/metal detector models based on user inputs (terrain, budget, experience). Built as a single static HTML page and published to GitHub Pages.

**Live demo:** https://<GITHUB_USERNAME>.github.io/gogold/

---

## What this repo contains

- `index.html` — the single static page (the recommender UI).
- `images/` (optional) — any local images you add.
- `README.md`, `LICENSE`, `.gitignore`.

---

## How to update the site

### Edit via GitHub web UI
1. Open the file in the repo on GitHub
2. Click the pencil icon to edit, then commit the change.
3. GitHub Pages will serve the updated `index.html` automatically (usually within 1 minute).

### Update locally (git)
```bash
# edit index.html locally
git add index.html
git commit -m "Update site"
git push origin main
